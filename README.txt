Solucion a la prueba tecnica para el proceso de contratacion de Procces OnLine.

Se adjunta:

1. Proyecto de Netbeans.
2. Modelo de la Base de Datos (MySQL).

Proceso de ejecucion:

1. iniciar el servicio MySQL en WAMP con el puerto 3305
2. Abrir modelo de BD en MySQL Workbench
3. Hacer forward engineer al modelo para crear la BD
4. Abrir el Proyecto en NetBeans
5. Ejecutar el Proyecto
